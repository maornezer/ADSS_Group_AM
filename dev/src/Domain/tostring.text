1. פונקציות המייצרות מחרוזת מידע על משלוח או אתרים
toString פונקציות
public String toString():

מטרה: יצירת מחרוזת המתארת את המשלוח בצורה כללית.
תיאור: מייצרת מחרוזת הכוללת מידע על כל אתר במשלוח ומשקל לאחר עזיבת כל אתר.
public String toStringWithInfo(double maxTruckWeight, double maxDriverWeight):

מטרה: יצירת מחרוזת המתארת את המשלוח עם מידע נוסף על מגבלות משקל.
תיאור: כוללת אזהרות אם המשקל חורג מהמקסימום המותר למשאית ולנהג.
public String toShortString():

מטרה: יצירת מחרוזת קצרה המתארת את המשלוח.
תיאור: מייצרת מחרוזת קצרה עם מידע על משקל המשלוח לאחר עזיבת כל אתר.
siteShippingToString פונקציות
public String siteShippingToString(Site site):

מטרה: יצירת מחרוזת המתארת את האתר והפריטים בו.
תיאור: כוללת מידע על האתר והפריטים המועברים אליו/ממנו. כוללת אזהרות אם הנתונים חלקיים או האתר לא תקין.
public String siteFullShippingToString(Site site, double maxWeight, double maxDriver):

מטרה: יצירת מחרוזת מפורטת על האתר והפריטים בו, כולל מגבלות משקל.
תיאור: כוללת אזהרות על חוסר בפריטים, משקל יתר לנהג או למשאית.
public String siteShippingToStringWithInfo(Site site, double maxTruck, double maxDriver):

מטרה: יצירת מחרוזת עם מידע על האתר והפריטים בו, כולל מגבלות משקל.
תיאור: דומה ל-siteFullShippingToString, עם אזהרות על חוסר בפריטים ומשקל יתר.
2. פונקציות ליצירת מחרוזות של פריטים ומשקל
public String ItemsToString():

מטרה: יצירת מחרוזת המתארת את כל הפריטים במשלוח.
תיאור: כוללת את מזהה הפריט, שמו ומשקלו.
public String itemInfoToString(Site site, String itemName):

מטרה: יצירת מחרוזת עם מידע על פריט מסוים באתר מסוים.
תיאור: כוללת שם הפריט, הכמות, משקלו ומשקל המשלוח הכולל.
public String getItemsStringWithAmount():

מטרה: יצירת מחרוזת עם מידע על כל הפריטים כולל הכמות הכוללת שנאספה.
תיאור: כוללת שם הפריט, משקלו והכמות הכוללת שנאספה במשלוח.
public String shipmentItemString(String itemName):

מטרה: יצירת מחרוזת עם מידע על פריט מסוים בכל אתרי המשלוח.
תיאור: כוללת מידע על הפריט ומשקלו בכל אתר.

public String getFullSiteString(int destOrder, double maxWeight):

public String getErrorsAtSite(int order, double truckMax, double truckWeight, double driverMax):

תיאור: כוללת אזהרות על חוסר בפריטים ומשקל יתר למשאית ולנהג.

public String[][] getSiteItemsInfo(int order):

מטרה: יצירת מערך דו-ממדי עם מידע על הפריטים באתר מסוים במשלוח.

תיאור: כוללת מידע על הפריטים ומשקל יתר למשאית.
private String getChangedItemsAmount(int order):

מטרה: יצירת מחרוזת עם סיכום השינויים בכמות הפריטים באתר מסוים.
תיאור: כוללת את כמות הפריטים שנאספה ונמסרה.
ניתוח כפילויות והכרחיות
כפילויות: ישנן פונקציות עם מטרות דומות כמו siteFullShippingToString, siteShippingToString, ו-siteShippingToStringWithInfo. ניתן לשקול לאחד אותן לפונקציה אחת עם פרמטרים נוספים כדי להפחית את הכפילויות.
הכרחיות: כל הפונקציות מספקות מידע שונה אך ייתכן שניתן לפשט ולייעל חלק מהן על ידי איחוד פונקציות דומות והוספת פרמטרים לבקרת התנהגות.
לדוגמה:

ניתן לאחד את הפונקציות siteShippingToString, siteFullShippingToString ו-siteShippingToStringWithInfo לפונקציה אחת עם פרמטרים שמאפשרים לכלול מידע נוסף לפי הצורך.
פונקציות כמו toStringWithInfo ו-toShortString יכולות להיות חלק מ-toString עם פרמטרים שמאפשרים לכלול מידע נוסף או קיצור המידע.
באופן כללי, רצוי לבדוק אילו פונקציות באמת נחוצות ולהסיר פונקציות מיותרות או למזגן, מה שיכול לפשט את התחזוקה של הקוד ולהפוך אותו ליעיל יותר.
